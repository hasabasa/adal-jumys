# Adal Jumys

Қазақстан еңбек нарығының ашықтық платформасы: жұмыс беруші рейтингтері, верификацияланған отзывтар, вакансия-шағымдар.

Ашық кодты жоба (AGPL-3.0). Идея және жоба авторы: Хасенхан ([cube-dev.tech](https://cube-dev.tech)). Байланыс, ұсыныс, қатысу тілегі болса: Telegram [@baldween](https://t.me/baldween).

# Неге керек

Қазақстанда жұмысшы құқықтарының бұзылуы күнделікті құбылыс: жалақының төленбеуі немесе кешігуі, зейнетақы аударымдарының жасалмауы, ақысыз овертайм, вакансиядағы жалақы-алдау (жарияланғаны 700 000, собеседованиеде айтылатыны 150 000 + пайыз), қазақтілді кандидаттарды кемсіту. Мұндай оқиғалар әлеуметтік желі тредтерінде тарап, із-түссіз жоғалып кетеді.

Жұмыс беруші репутациясының орталық, дәлелге сүйенетін алаңы елде жоқ. Adal Jumys осы олқылықты толтырады: әр шағым фактімен, дәлелмен және құрылымды формада жазылады, компанияға жауап беру құқығы қаралады, модерация ашық жүреді.

# Не істейді (v1)

- Компания каталогы: БСН бойынша ресми реестрден автотолтырылатын карточкалар
- Жұмысшы отзывтары: баға 1-10, категориялар (жалақы уақтылығы, зейнетақы аударымдары, овертайм, еңбек шарты), дәлел-файлдар
- Вакансия-шағымдар: құрылымды форма (жалақы-алдау, фейк-вакансия, ghost-вакансия, төленбейтін тестілік, кемсітушілік және т.б.)
- Кемсітушілік блогы: тіл, жас, жыныс, ұлт бойынша, дәлел түрлерімен
- Екі бөлек ось: жұмыс беруші рейтингі (қызметкер отзывтары, сандық) және жалдаушы көрсеткіші (кандидат шағымдары, статистика + бейджтер)
- Компанияның ресми жауап беру құқығы (right of reply)
- Ашық модерация: барлық модераторлық әрекет жария аудит-логта

# Ұстанымдар

1. **Qazaq-first.** Интерфейстің әдепкі тілі қазақша, орысша толық қолжетімді.
2. **Факт, эмоция емес.** Платформа нақты фактіге итереді: "жалақы 3 ай кешіктірілді, скрин тіркелген". Бағалауыш қорлау модерациядан өтпейді.
3. **Дербес деректер қорғалады.** Жеке адамдардың аты-жөні, телефоны ешқашан жарияланбайды: тек компанияның ресми жалпыға ашық контактілері. Верификация құжаттары модератор шешімінен кейін бірден өшіріледі.
4. **Қажетсіз дерек жиналмайды.** Тіркелуге телефон номері сұралмайды, аты-жөні де сұралмайды: тек email және псевдоним. Отзывтар жария түрде псевдониммен көрсетіледі. Платформа сақтамайтын деректі ешкім одан алып та кете алмайды.
5. **Ашықтық.** Код ашық, рейтинг формуласы ашық және документтелген, модерация аудит-логы жария.
6. **Рейтинг сатылмайды.** Ешқандай ақылы қызмет компанияның рейтингіне әсер ете алмайды.

# Техникалық стек

- Backend: FastAPI, PostgreSQL (asyncpg), SQLAlchemy 2, Alembic, Redis, uv
- Frontend (жоспарда): Next.js (App Router), TypeScript, Tailwind, i18n kk/ru
- Дәлел-файлдар: S3-үйлесімді қойма
- Деплой: Docker Compose

# Жергілікті іске қосу

Талап: [uv](https://docs.astral.sh/uv/) және Docker (немесе жергілікті PostgreSQL 15+ пен Redis).

```bash
git clone https://github.com/hasabasa/adal-jumys.git
cd adal-jumys

# PostgreSQL + Redis
docker compose up -d

# Орта айнымалылары (мәндерін өз ортаңа сай толтыр)
cp .env.example .env

# Backend
cd backend
uv sync
uv run alembic upgrade head
uv run uvicorn app.main:app --reload
```

Тексеру: `http://localhost:8000/health` және `http://localhost:8000/docs` (Swagger).

Контрибьюторларға: коммит алдында `pre-commit install` жүргізіп ал (gitleaks құпия-сканы және базалық тексерулер).
