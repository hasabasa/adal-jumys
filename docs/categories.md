# Категориялар каталогы (заң-негізімен)

Платформадағы әр бұзушылық-категория ҚР заңнамасының нақты нормасына байланған. Бұл (а) шағымды дауға берік қылады, (б) болашақ Шағым-шебердің (v2) дайын негізі: категория белгілі болса, ресми өтініштегі бап-сілтеме де белгілі.

Негізгі дереккөз: ҚР Еңбек кодексі, 2015 ж. 23 қараша, N 414-V (төменде "ЕК").

# Қолдану ережесі

- Кестедегі бап-сілтемелер юзерге формада көрсетілмейді (форма қарапайым қалады), бірақ категория-түсіндірмеде және болашақ Шағым-шеберде қолданылады.
- "Әлеуметтік категория" деген - нақты бабы жоқ, бірақ қоғам үшін маңызды тәжірибе-белгі. Олар да жиналады, статистикада көрінеді.
- Бап нөмірі өзгерсе (кодекс жаңарса), осы файл жаңартылады.
